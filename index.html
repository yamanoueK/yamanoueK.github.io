こんにちは。山野上健一と申します。

また、社内の新人教育も担当し積極的に新しいコンテンツを導入していきました。

大概の事は、「できる or できない」ではなく「やる or やらない」と思っています。

私自身のクリエイターとしての基本理念は･･･

価値あるものとは「ユーザーに求められるものである」を基本理念としています。

実績(SINCE 2016/04/～)

※業務の都合で、実際の画面イメージお見せすることはできません。

  • 社内便管理システム

    HTML5 CSS3 JavaScript PHP

    「モダンなデザインで」という要望により、BootStrap3を使用。レスポンシブデザインを入れると共に、Excelに近い操作でI/Oができるように開発。

  • 設備保全システム

    HTML5 CSS3 JavaScript PHP

    「保全のコストを軽減したい」という課題から、新入社員でも一定水準の保全ができるようにオペレーションを意識して開発。

    RFIDタグを読み込み、点検結果をHTTP通信でWEBサーバーへ登録する事で、どこからでも各設備の点検結果を確認できるように開発。

  • 測定結果管理システム

    HTML5 CSS3 JavaScript PHP

    Ajax通信を用いて、リアルタイムでデータを表示するように開発。また、画面遷移を極力なくすためにモーダルウインドウ等、jQueryを積極的に使用。

  • 在庫管理/受発注管理システム

    VB.NET

    4人チームのプロジェクトリーダーとして参加。

    プロジェクトマネジメントにおいて、メンバーが作業しやすいように、念入りに仕様書やDB定義書のメンテナンスを行った。

    開発作業時も、「使い回し」を意識した作成をするように心がけた。

  • レセプト管理システム

    VB.NET

    従来の作業と共存できるシステムを作成。

    ユーザー様から「月末のレセプト処理が非常に楽になり、ミスが減った」と感想をもらった。

  • ファイル送信アプリ(Android)

    JAVA

    セキュアなファイル転送を行う必要があったため、SCP通信を用いたファイル転送アプリを作成。

    Vagrat＋VMを用いて、バーチャルにCentOSを構築し開発を行った。
